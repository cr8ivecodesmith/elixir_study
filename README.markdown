Elixir Lang Study
=================

Code and notes for studying Elixir


#### Requires

- Docker
- Docker Compose
- Elixir 1.2+


#### Running

Run Elixir's interactive shell

```
docker-compose run --rm iex
```


Run a bash shell

```
docker-compose run shell
```


Run a once off script

```
docker-compose run --rm shell elixir src/hello1.exs
```
